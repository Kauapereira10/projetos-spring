# 🚀 Spring Boot Practice Projects

Este repositório contém uma coleção de **mini projetos práticos** desenvolvidos com o objetivo de estudar e evoluir minhas habilidades em desenvolvimento backend com **Spring Boot**.

Aqui você encontrará implementações simples, mas fundamentais, cobrindo conceitos importantes como APIs REST, arquitetura MVC, persistência com JPA e muito mais.

---

## 📚 Objetivo

Este repositório foi criado para:

* Praticar conceitos do ecossistema Spring
* Evoluir na construção de APIs REST
* Entender boas práticas de arquitetura
* Servir como portfólio de aprendizado contínuo

---

## 🛠️ Tecnologias Utilizadas

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Hibernate
* H2 Database / MySQL
* Maven

---

## 📂 Estrutura do Repositório

Cada pasta representa um projeto independente:

```
📦 projetos-spring
 ┣ 📂 todolist-api
 ┣ 📂 nome do projeto
 ┣ 📂 nome do outro projeto
 ┗ 📂 ...
```
